# Ronini.dots

These are my personal dotfiles. They are a collection of scripts, configurations, and tools that I use to set up my development environment.

## Getting Set Up

1. Clone the repo
```bash
git clone git@github.com:Roninii/dots.git
```
2. Run the setup script
```bash
cd dots
chmod +x setup.sh
./setup.sh
```
